## [1.0.0] - 2026-02-12

### Added
- **5 Complete Stack Templates**:
  - Next.js 14+ App Router + Tailwind CSS
  - Python FastAPI with async/await patterns
  - React Native + Expo with Expo Router
  - TypeScript strict mode (framework-agnostic)
  - Go microservices with Gin framework

- **Comprehensive Prompt Library** (30+ prompts):
  - Refactoring prompts (8 prompts)
  - Testing prompts (7 prompts)
  - Documentation prompts (7 prompts)
  - Debugging prompts (8 prompts)
  - API/Backend prompts (8 prompts)

- **VS Code Optimization**:
  - settings-snippet.json with AI-friendly configurations
  - keybindings-snippet.json for Cursor shortcuts
  - extensions-list.txt with recommended extensions

- **Documentation**:
  - README.md with installation instructions
  - LICENSE.txt defining usage terms
  - PLAN.md with marketing strategy
  - HLD.md (High-Level Design)
  - LLD.md (Low-Level Design)

### Notes
- Initial release
- Tested on Next.js 14, Python 3.11, React Native with Expo SDK 50
- All files UTF-8 encoded
- Total package size: ~45KB (uncompressed)

### Added (v1.0.1 - Day 1 Patch)
- **Refined Master Prompts**: Upgraded "Clean Component Generator" to "Senior Architect" standards with strict constraints.
- **Documentation Updates**: Added verification steps to README.md to help users confirm rules are active.
- **Fail-Safe Warnings**: Added critical warnings about JSON merging in VS Code settings.
